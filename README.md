# NeutronAdapter
A semantic client to consume and publish neutron relevations data on a SEPA database
By Greg and Franceso

# How to use
## Prerequisites
- node
- tsc

## How to use
1. Download the required packages
```
npm i
```

2. Build
```
npm run build
```

3. Start
```
npm start
```

4. (Optional) run in watch mode
```
npm run dev
```
