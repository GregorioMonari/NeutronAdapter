# NeutronAdapter
A semantic application that extracts soil moisture from neutron data provided by CNRS sensors.


# How to use
## Prerequisites
- node
- tsc

## Run the application
1. Download the required packages
```
npm i
```

2. Build
```
npm run build
```

3. Start
```
npm start
```

4. (Optional) run in watch mode
```
npm run dev
```
